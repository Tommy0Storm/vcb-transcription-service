Its format for filing in court is governed by the Practice Directives (also known as Practice Manuals) of the specific High Court Division in which it will be used (e.g., Gauteng, KwaZulu-Natal).14The de facto "Golden Rule" for document typography, synthesized from various practice directives, provides the standard for a 100% compliant document. A notable example from a provincial practice manual (which reflects a national standard for legal process) states:"All legal processes filed with the court must be typed or printed on standard A4 paper; in font size 12 point; with theme fonts either Arial or Times-Roman; line spacing set at 1.5, and with margins of at least 2 cm on all sides of the page.".17Therefore, all references to "Rule 8" in the VCB-Transcription template must be removed. This function assumes a perfect 1:1 correspondence in the number of segments between the English and Afrikaans sections. As seen in the provided text file 9, a single English segment may be translated into multiple Afrikaans segments (or vice-versa), or a speaker may have multiple lines of text for a single timestamp. The zip logic is brittle and will fail.A robust, production-ready solution must use the timestamp as the unique key to join the data. The following Python parser is resilient to segment-order and count mismatches. It reads the file once, identifies language blocks, and maps all data to its corresponding timestamp.3.2 The VCB-Transcription Master Generation ScriptThe following is the complete Python script to generate both the professional and court-compliant documents. def parse_bilingual_file_robust(filepath):
    """
    Parses the bilingual transcription file using a robust, timestamp-keyed
    dictionary. This avoids errors caused by misaligned segments.
    """
    with open(filepath, 'r', encoding='utf-8') as f:
        content = f.read()

    # Regex to find speaker and timestamp
    # e.g., "SPEAKER 1 [00:00:00]"
    speaker_regex = re.compile(r'^(SPEAKER \d+) (\[\d\d:\d\d:\d\d\])$')
    
    parsed_data = {}
    current_lang_prefix = None
    current_timestamp = None

    for line in content.split('\n'):
        line = line.strip()
        if not line:
            continue

        # Detect language change
        if line.lower().startswith('# english transcription'):
            current_lang_prefix = 'en'
            continue
        elif line.lower().startswith('# afrikaans transcription'):
            current_lang_prefix = 'af'
            continue

        if not current_lang_prefix:
            continue

        match = speaker_regex.match(line)
        if match:
            # New speaker/timestamp segment
            speaker = match.group(1)
            timestamp = match.group(2)
            current_timestamp = timestamp
            
            # Ensure the timestamp key exists
            if timestamp not in parsed_data:
                parsed_data[timestamp] = {}
            
            # Store speaker and initialize text
            parsed_data[timestamp][f'{current_lang_prefix}_speaker'] = speaker + ":"
            parsed_data[timestamp][f'{current_lang_prefix}_ts'] = timestamp
            if f'{current_lang_prefix}_text' not in parsed_data[timestamp]:
                parsed_data[timestamp][f'{current_lang_prefix}_text'] =

        elif current_timestamp and current_lang_prefix:
            # This is a text line belonging to the current speaker
            # Append the text to the correct language entry
            key = f'{current_lang_prefix}_text'
            if key in parsed_data[current_timestamp]:
                parsed_data[current_timestamp][key].append(line)
            else:
                # This can happen if text appears before a speaker (e.g., file header)
                # We will ignore it.
                pass

    # Process the collected data into the final list format
    # We must sort by timestamp to ensure chronological order
    final_data_list =
    
    # Sort keys (timestamps) chronologically
    sorted_timestamps = sorted(parsed_data.keys(), key=lambda ts: ts.strip(''))

    for timestamp in sorted_timestamps:
        entry = parsed_data[timestamp]
        
        # Join text lines into a single string
        if 'en_text' in entry:
            entry['en_text'] = ' '.join(entry['en_text'])
        if 'af_text' in entry:
            entry['af_text'] = ' '.join(entry['af_text'])
            
        # Add default values for missing keys to prevent template errors
        entry.setdefault('en_speaker', '')
        entry.setdefault('en_ts', timestamp)
        entry.setdefault('en_text', '')
        
        entry.setdefault('af_speaker', '')
        entry.setdefault('af_ts', timestamp)
        entry.setdefault('af_text', '') # Empty string if no translation

        final_data_list.append(entry)

    return final_data_list

def generate_professional_transcript(context, output_filename):
    """Generates the 'Ultra Premium' professional transcript."""
    try:
        doc = DocxTemplate('VCB_Premium_Professional_Template.docx')
        doc.render(context)
        doc.save(output_filename)
        print(f"\n--- PROFESSIONAL SUCCESS ---")
        print(f"Document created: {output_filename}")
    except Exception as e:
        print(f"\n--- PROFESSIONAL ERROR ---")
        print(f"Failed to create professional transcript: {e}")
