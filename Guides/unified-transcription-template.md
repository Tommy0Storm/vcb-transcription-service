# PART 3: GOOGLE TRANSCRIPTION INTEGRATION STEPS

## Step 1: Enable and Export from Google

1. Upload your audio/video to Google Docs or use Google Recorder
2. Generate automatic transcription
3. **Export steps:**
   - Copy entire transcription
   - Paste into template
   - Review for accuracy (Google achieves ~85% accuracy)

## Step 2: Quick Clean-Up Tasks

**Remove:**
- ❌ [Music] / [Noise] markers (replace with proper notation below)
- ❌ Automated paragraph breaks (you'll control these)
- ❌ Confidence markers if present

**Replace with:**
- ✓ Proper speaker labels: [translate:Person Name]: instead of generic names
- ✓ Timestamps in format: [HH:MM:SS]
- ✓ Non-verbal notations: [PAUSE - X seconds], [INAUDIBLE], etc.

## Step 3: Map Google Output to Template

**Google gives you:**
```
[Timestamp] Speaker Name
Text content
```

**Convert to template format:**
```
[Timestamp]

**SPEAKER NAME:** (Bold)
[Your transcribed text from Google]
```

## Step 4: Validation

Use embedded validation checklist (PART 7) to verify:
- All speakers identified
- All timestamps present
- Special notations properly formatted
- Language consistency

# PART 6: FORMATTING CONVERSION SCRIPTS

## Script A: Converting Google Transcription to Professional Template

```
INPUT (Google Format):
[0:42] Speaker Name
This is the transcribed text that Google generated.

CONVERSION STEPS:
1. Add full timestamp: [00:00:42]
2. Bold speaker name: **SPEAKER NAME:**
3. Maintain text verbatim
4. Add spacing between speakers

OUTPUT (Professional Template):
[00:00:42]

**SPEAKER NAME:**
This is the transcribed text that Google generated.
```

## Script B: Converting Google Timestamps to High Court Format

```
Google Format: [0:42] (from context start)
Required: [HH:MM:SS] with leading zeros

CONVERSION:
- If 42 seconds → [00:00:42]
- If 5 minutes, 30 seconds → [00:05:30]
- If 1 hour, 23 minutes, 15 seconds → [01:23:15]

For High Court (double-spacing):
- Google output: "Text on one line"
- Convert to: "Text on one line with"
             "double spacing maintained"
```

## Script C: Professional to High Court Conversion

```
CHANGES REQUIRED:

1. LINE SPACING:
   Before: Single (1.0)
   After: Double (2.0)
   
2. LINE NUMBERING:
   Before: None / Optional
   After: Every 10th line (10, 20, 30, etc.)
   
3. PAGE HEADERS:
   Before: Optional
   After: MANDATORY - Include witness name
   
4. MARGINS:
   Before: 1 inch (may reduce to 0.75 for landscape)
   After: STRICT 1 inch (2.54cm) on all sides
   
5. CERTIFICATION:
   Before: Optional translator cert
   After: MANDATORY Rule 59 certification
```

# PART 8: LANGUAGE-SPECIFIC GUIDES (11 Official SA Languages)

## Language 1: [translate:English]

**Template Notes:**
- Standard template applies
- No special character requirements
- Timestamps: [HH:MM:SS] standard format

**Speaker Labels:**
- Use: **COUNSEL FOR PLAINTIFF:** or **WITNESS:**
- Formal legal terminology standard

**Example:**
```
[00:05:30]

**COUNSEL FOR PLAINTIFF:**
Please describe your experience with the transaction in question.

**WITNESS:**
The transaction occurred on January 15, 2024. I was present 
throughout the process.
```

## Language 2: [translate:Afrikaans]

**Template Notes:**
- Use South African Afrikaans spelling (not European)
- Formal vs. informal address: [translate:u] (formal) vs. [translate:jy] (informal)
- Court proceedings use formal [translate:u]

**Speaker Labels:**
- Use: **PROKUREUR VIR EISER:** / **GETUIE:**
- Or bilingual: **COUNSEL FOR PLAINTIFF / PROKUREUR VIR EISER:**

**Key Terms for Court:**
- [translate:Hof] = Court
- [translate:Getuie] = Witness
- [translate:Bewering] = Allegation
- [translate:Tydstip] = Timestamp

**Example:**
```
[00:05:30]

**PROKUREUR VIR EISER:**
Kan u asseblief u rol in die organisasie beskryf?

**GETUIE:**
Ek is die Finansiële Bestuurder. Ek beklee hierdie pos 
vir sewe jaar.
```

**Column Width Adjustment:**
- Afrikaans typically requires 10-15% more space than English
- Adjust table column width if needed in bilingual format

---

## Language 3: [translate:IsiZulu]

**Template Notes:**
- Native script with click consonants
- May require 20-30% more space than English
- Font must support IsiZulu characters

**Speaker Labels:**
- Use: **UMLAWULI WENTAWULO / UMANGALALI:**
- Or: **WITNESS / UMANGALALI:**

**Formatting Considerations:**
- Ensure proper rendering of: [translate:hl], [translate:c], [translate:ph], [translate:x] (click consonants)
- Font recommendation: Arial Unicode MS or Calibri
- Line height may need adjustment for diacritics

**Key Terms:**
- [translate:Inkhosi yeNkantolo] = Court
- [translate:Umangalali] = Witness
- [translate:Isithi] = Statement

**Example:**
```
[00:05:30]

**UMLAWULI WENTAWULO:**
Ngiyakucela ucace indlela ozibandakanye ngayo noshishino.

**UMANGALALI:**
Ngingu-Mgcinimani wezikhwefu. Nbakhe lesi sikhundla 
iminyaka eyisikhombisa.
```

# PART 10: TROUBLESHOOTING GUIDE

## Common Google Transcription Issues

| Issue | Cause | Solution |
|---|---|---|
| Speaker not identified | Multiple speakers, similar voices | Manually label speakers before uploading |
| Timestamps missing | Recording too long | Add timestamps manually at key points |
| Words missing/corrupted | Poor audio quality | Re-record or use highest quality version |
| Wrong speaker attribution | Background noise | Edit and correct in template |
| Text too compressed | Auto-formatting | Apply proper spacing manually |

## Common Template Issues

| Issue | Cause | Solution |
|---|---|---|
| Bilingual columns misaligned | Manual text entry errors | Use table auto-fit and verify timestamps |
| Line numbering jumps | Manual errors | Use Word's automatic line numbering feature |
| Double-spacing breaks | Single lines inserted | Review each paragraph for spacing |
| Translator info missing | Incomplete header | Use PART 5 field variables |
| Certification page errors | Wrong template | Reference correct certification text |
